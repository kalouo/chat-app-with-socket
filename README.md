# chat-application

##. About

This is a basic chat application set up using socket.io, mongoose (ODM for MongoDB) and Vue.

## Project setup

```
yarn install
```

### Compiles and minifies for production

```
yarn build
```

### Fires up the server on localhost:3001

```
yarn start
```
